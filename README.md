## Running exercises locally

Install the toolchain for the language track you want to run. The repository contains
solutions across Ruby, Elixir, Rust, Go, and Erlang, so you only need the runtime for
the specific exercise you are checking.

Most folders follow the structure generated by the Exercism CLI. To run an exercise locally, open its directory and use the test command for that language track, for example:

```bash
# Ruby
ruby *_test.rb

# Elixir
mix test

# Rust
cargo test

# Go
go test ./...
```

The exact command can vary by exercise depending on the files generated by Exercism.
Some archived solution folders contain only the submitted implementation file; if tests are missing,
re-download the exercise with the Exercism CLI to restore the full test scaffold.
