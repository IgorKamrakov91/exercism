# Exercism Solutions

This repository contains my personal solutions for exercises completed on [Exercism](https://exercism.org/).

The code is organized by language track and exercise name under `solutions/`. Each exercise may contain one or more iterations as the solution is refined over time.

```text
solutions/
  ruby/<exercise>/<iteration>/

  go/<exercise>/<iteration>/
  erlang/<exercise>/<iteration>/
```

## Learning goals

- Practice idiomatic Ruby and Elixir through small, focused problems.
- Compare different approaches across language tracks.
- Keep a public record of progress and refactoring iterations.
- Revisit older exercises when learning a cleaner or more expressive solution.

## Running exercises locally

Most folders follow the structure generated by the Exercism CLI. To run an exercise locally, open its directory and use the test command for that language track, for example:

```bash
# Ruby
ruby *_test.rb

# Elixir
mix test

# Rust
cargo test

# Go
go test ./...
```

The exact command can vary by exercise depending on the files generated by Exercism.
